Detailed Description

Class used to represent a gain function.

A gain function is a function from a set of secrets and a set of adversary actions to a real number.

A gain makes reference to an adversary's gain. The adversary can take some action, and for each secret, he or she achieves a gain.

A gain function makes reference to a set of secrets, so one of the attributes of this class is a pointer to a Distribution, which represents the prior distribution over the set of secrets.

Construtor used when the gain function matrix is in a file.

The file must be in the following format: 

w n
p11 p12 ... p1n
p21 p22 ... p2n
...
pw1 pw2 ... pwn

where w is the number of actions, and n is the number of secrets. The rows correspond to actions and the columns correspond to secrets. Each probability pij is the gain of taking action i when the secret value is j. Every two numbers must be separated by a " " (space).

Print the gain function matrix in a file.

The gain function matrix is printed in the following format: 

w n
p11 p12 ... p1n
p21 p22 ... p2n
...

where w is the number of actions, and n is the number of secrets. Each probability pij is the gain of taking action i when the secret value is j. Every two numbers are separated by a " " (space).

◆ gain

std::vector<std::vector<long double> > Actions::gain

Gain function matrix.

The lines are the actions, the columns are the secrets and each cell (w,x) in the matrix is the gain that an adversary achieves by taking action w when the actual value of the secret is x.

The matrix has dimensions W x N, where W is the number of actions (num_ac) and N is the number of secrets (num_el of the prior).
