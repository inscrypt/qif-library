Detailed Description

Class used to represent a probability distribution.

A probability distribution is on a set of elements, so the class has the attribute num_el to keep the number of elements in the set and the attribute prob to keep the probability of each element from the set occurs.

Constructor used to generate a uniform or a random probability distribution.

Parameters
num_elNumber of elements in the set of elements.
typeMust be "uniform" for a uniform distribution or "random" for a random distribution.
max_probA number in the interval [0,1]. When type is "uniform", max_prob is ignored (any value can be passed). When type is "random" it is used as an upper bound for generating the elements' probabilities in the distribution. For example, when max_prob is 0.4, all the probabilities in the new probability distribution are less equal to 0.4.

Check if a vector of numbers make up a probability distribution.

  • For every probability p in the vector, 0 <= p <= 1;
  • The sum of all probabilities in the vector is 1 , with margin of error equals to the consant EPS (upper or lower) - see the definition of constant EPS.

std::vector<long double> Distribution::prob

A vector of elements' probabilities.

It is a vector which contains all the elements' probabilities. Each position i in the vector is the probability of the ith element occurs. The vector is indexed from the position 0 (1st element) to the position num_el - 1 (last element). So the vector's size is num_el.
