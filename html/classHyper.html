Detailed Description

Class used to represent a hyper-distribution.

A hyper-distribution is a distribution on distributions on a set of secrets. Each output in a informational channel is a possible "world", and each possible world is a new distribution on the set of secrets. We call the possible worlds the inner distributions. Each possible world has a probability to occur, and we call the distribution on the possible worlds as the outer distribution.

We can calculate the outer and the inners distributions using the joint distribution between the set of secrets and the set of outputs.

void Hyper::printToFile ( std::string  type,
std::string  file,
int  precision 
)

Print the joint matrix, outer distribution or the inners distributions in a file.

The choice of what will be printed is passed in the first parameter type.

The joint matrix is printed in the following format: 

n y
p11 p12 ... p1y
p21 p22 ... p2y
...
pn1 pn2 ... pny

where n is the number of secrets and y is the number of outputs of the channel. Each pij is the joint probability p(i,j), for a secret i and an output j.

The outer distribution is printed in the following format: 

y
p1 p2 ... py

where n is the number of outputs in the channel. Each pi is the probability of the ith output happens.

The inners' matrix is printed in the following format: 

n y
p11 p12 ... p1y
p21 p22 ... p2y
...
pn1 pn2 ... pny

where n is the number of secrets and y is the number of outputs of the channel. Each pij is the conditional probability p(i|j) which is the probability of the secret being i when the output is j.

Each two numbers in any printing are separated by a " " (space).

Parameters
typeThe name of what will be printed. It must be "joint", "outer" or "inners", to print the joint matrix, outer distribution or the inners distribution matrix, respectively.
fileA file's name.
precisionDecimal precision for float numbers. For example, use the value 3 to print 0.322, use 5 to print 0.32197, and so on.
