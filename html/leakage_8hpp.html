Gain &  gain

The relative difference between the posterior g-vulnerability and the prior g-vulnerability.

Parameters
hyperA hyper-distribution.
gainA gain function for a set of secrets.
Returns
If gVulnerability(prior distribution) is 0 and gVulnerability(hyper-distribtuion) is also 0, it returns 1.
If gVulnerability(prior distribtuion) is 0 and gVulnerability(hyper-distribtuion) is different of 0, it returns POSITIVE_INFINITE.
If both gVulnerability(prior distribtuion) and gVulnerability(hyper-distribtuion) are different of 0, it returns gVulnerability(hyper-distribtuion) / gVulnerability(prior distribtuion).

See the section 5.2 in the reference [1] for more details about the return of multiplicative leakage.
