Detailed Description

Class used to represent a gain function.

A gain function is a function from a set of secrets and a set of adversary actions to a real number.

A gain makes reference to an adversary's gain. The adversary can take some action, and for each secret, he or she achieves a gain.

As a gain function makes reference to a set of secrets, one of the attributes is a pointer to a Distribution, which represents the prior probability distribution on the set of secrets.

Construtor used when the gain function matrix is in a file.

The file must be in the following format: 

w n
p11 p12 ... p1n
p21 p22 ... p2n
...
pw1 pw2 ... pwn

where w is the number of actions, and n is the number of secrets (n = Distribution::num_el in the prior). The rows correspond to actions and the columns correspond to secrets. Each probability pij is the gain of taking action i when the secret value is j. Each two numbers must be separated by a " " (space).

Parameters
priorPrior distribution on a set of secrets
fileFile's name which contains a gain function matrix.
Warning
The number of rows in the gain function matrix must be as same as the number of elements in the prior distribution.

The set of secrets can be described by a Distribution, so it is the first parameter.

Parameters
priorPrior distribution on a set of secrets.
num_actNumber of actions.
MINLower bound for the gain of an action.
MAXUpper bound for the gain of an action.

The parameters MIN and MAX must form an interval. All the gains generated randomly will be in the interval [MIN,MAX]. Each gain is a long double value.

Warning
The number of rows in the gain function matrix must be as same as the number of elements in the prior distribution.

Print the gain function matrix in a file.

The gain function matrix is printed in the following format: 

w n
p11 p12 ... p1n
p21 p22 ... p2n
...

where w is the number of actions, and n is the number of secrets. Each probability pij is the gain of taking action i when the secret value is j. Each two numbers are separated by a " " (space).

Parameters

◆ matrix

std::vector<std::vector<long double> > Gain::matrix

Gain function matrix.

The lines are the actions, the columns are the secrets and each cell (w,x) in the matrix is the gain that an adversary achieves by taking action w when the actual value of the secret is x.

The matrix has dimensions W x N, where W is the number of actions (num_act) and N is the number of secrets (num_el of the prior).
