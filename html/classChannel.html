Detailed Description

Class used to represent an informational channel.

A channel can be described as a system that takes as input a secret - from a set of secrets - and whose only observable behavior is to produce an output - from a set of possible outputs. So a channel is a function from a set of secrets to a set of possible outputs.

We can use a matrix to represent the entire channel. In this library, a channel matrix is a matrix whose rows give the distribution on outputs corresponding to each possible input. Hence each cell is the conditional probability p(y|x), for every secret x in the set of secrets and every y in the set of possible outputs.

As a channel makes reference to a set of secrets, one of the attributes is a pointer to a Distribution, which represents the prior probability distribution on the set of secrets.

Channel::Channel ( Distribution &  prior,
std::string  file

Construtor used when the channel matrix is in a file.

The file format must be: 

n y
p11 p12 ... p1y
p21 p22 ... p2y
...
pn1 pn2 ... pny

where n is the number of secrets (n = Distribution::num_el in the prior) and y is the number of outputs. The rows correspond to secrets and the columns correspond to outputs. Each pij is the conditional probability p(j|i) of the output j happens when the current secret is i. Each two numbers must be separated by a " " (space).

The rows give the distribution on outputs corresponding to each possible input.

Parameters
priorPrior distribution on a set of secrets
fileFile's name which contains a channel matrix.
Warning
The number of rows in the channel matrix must be as same as the number of elements in the prior distribution.

Constructor used to generate random channels.

Given a set of secrets in the pointer prior, the constructor builds a random channel matrix. The number of the channel outputs is passed as an argument.

Parameters
priorPrior distribution on a set of secrets.
num_outNumber of outputs.
max_probA float number in the interval [0,1]. It is used as an upper bound of any outputs' probabilities. For example, when max_prob is 0.4, all the probabilities in the matrix will be less equal to 0.4.
Warning
We must have max_prob * num_out >= 1, otherwise a row in the channel matrix would not represent a probability distribution.
The number of rows in the channel matrix must be as same as the number of elements in the prior distribution.

Print the channel matrix in a file.

The channel matrix is printed in the following format: 

n y
p11 p12 ... p1y
p21 p22 ... p2y
...
pn1 pn2 ... pny

where n is the number of secrets (n = Distribution::num_el in the prior) and y is the number of outputs of the channel. Each pij is the conditional probability p(j|i) of the output j happens when the current secret is i. Each two numbers are separated by a " " (space).

Parameters
fileA file's name to print the channel matrix.
precisionDecimal precision for float numbers. For example, use the value 3 to print 0.322, use 5 to print 0.32197, and so on.
