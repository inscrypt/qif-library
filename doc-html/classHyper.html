Definition of hyper-distribution:

If X is a finite set (of possible secret values), π is the prior distribution on X, and C is a channel, a hyper-distribution [π>C] resulted from C on π is a distribution on distributions on X. Each output in an informational channel is a possible "world", and each possible world is a new distribution on the set of secrets. We call the possible worlds the inner distributions. Each possible world has a probability of occurring, and we call the distribution on the possible worlds as the outer distribution.

A Hyper object has 7 attributes:

  • prior : A pointer to a Distribution object, which represents the prior probability distribution on the set of secrets.
  • channel : A pointer to a Channel object.
  • joint : Joint dstribution matrix, where each position joint[x][y] is the joint probability p(x,y).
  • inners : The inner distributions matrix. It is the set of possible "worlds" of a hyper-distribution. Each column of this matrix is a posterior distribution on the set of secrets.
  • outer : The probability distribution on the inners.
  • num_post : Number of posterior distributons.
  • labels : A map from an integer to a set. The number of posterior distributions can differ from the number of channel outputs, because when we have two or more posteriors that are equals, we simplify them merging into a single posterior, and sum their outer probabilities. So labels maps from posterior distributions to a subset of channel outputs.

Function documentation

Hyper::Hyper()

Default constructor.

This constructor creates a Hyper object with prior and channel equals to NULL, num_post = 0 and joint matrix, inners matrix, outer and labels empty.

Hyper::Hyper(std::string prior_file, std::string channel_file)

Constructor used when the prior distribution on the set of secrets and the channel are in a file.

Parameters
prior_file File name that contains a probability distribution.
channel_file File name that contains a channel matrix.

The file format of prior must be the following:

n
p1
p2
...
pn

where n is the number of elements in the distribution and pi is the probability of the ith element occurs.

The file format of channel must be the following:

n y
p11 p12 ... p1y
p21 p22 ... p2y
...
pn1 pn2 ... pny

where n is the number of secrets and y is the number of outputs. Each pij denotes p(j|i), the conditional probability of getting output j given input i. Each two numbers must be separated by a space (" ").

Warning

The number of rows in the channel matrix must be equals to the number of elements in the prior distribution.

std::string Hyper::toString(std::string type, int precision)

Generates a string with the joint matrix, the outer distribution, the inner distributions or the labels.

Parameters
type The name of what will be returned. It must be "labels" to return the labels, "joint", "outer" or "inners", to return the joint matrix, outer distribution or the inners' matrix, respectively.
precision Decimal precision for float numbers. For example, use the value 3 to print 0.322, use 5 to print 0.32197, and so on.
Returns Possible returns:

  • The joint matrix is returned in the following format:

    p11 p12 ... p1y
p21 p22 ... p2y
...
pn1 pn2 ... pny

    where n is the number of secrets, y is the number of channel outputs and pij is the joint probability p(i,j).

  • The outer distribution is returned in the following format:

    p1 p2 ... py

    where y is the number of outputs.

  • The inners matrix is returned in the following format:

    p11 p12 ... p1y
p21 p22 ... p2y
...
pn1 pn2 ... pny

    where n is the number of secrets, y is the number of outputs and pij is the conditional probability p(i|j).

  • The labels are returned in the following format:

    1: x11 x12 ...
2: x21 x22 ...
...
3: xn1 xn2 ...

    where n is the number of posteriors (num_post) and xij is an integer between 0 and the number of outputs in channel.

Each two numbers in any choice are separated by a space (" ").

Distribution* Hyper::prior

std::vector<std::vector<long double>> Hyper::joint

Joint distribution matrix.

It is a matrix n x y where n is the number of secrets in channel and y is the number of outputs in channel. Each value joint[i][j] is the joint probability p(i,j). The joint distribution is build multiplying prior distribution by each row of the channel matrix.

The rows and columns are indexed from the position 0 (1st element) to the position prior ->num_el-1 (for rows) or num_out-1 (for columns).

Distribution Hyper::outer

A distribution on posterior distributions.

Each value outer[i] is the probability p(i) of the ith posterior distribution occuring.

The vector is indexed from the position 0 (1st element) to the position num_post - 1 (last element). So the vector size = num_post.

int Hyper::num_post

Number of posterior distributions.

It is the number of posterior distributions yielded by channel and prior.

Warning

The number of posterior distributions can be different from the number of outputs in channel. See labels for more details.

std::vector<std::vector<long double>> Hyper::inners

A matrix for the inner distributions.

It is a matrix n x y where n is the number of secrets in channel and y is the number of posterior distributions (num_post). Each value inners[i][j] is the conditional probability p(i|j), that is, the probability of the secret being i when the output is j. Each column is a posterior distribution.

The rows and columns are indexed from the position 0 (1st element) to the position prior ->num_el-1 (for rows) or num_post-1 (for columns).

Warning

The number of columns in inners (that is, the number of posterior distrbutions) can be different from the number of outputs in channel. See labels for more details.

std::map<int, std::set<int>> Hyper::labels

A map from posterior dsitributions indexes to a set of channel outputs indexes.

When we look at the hyper-distrubtion, we have the outer distribution and the inner distributions. Depending on the prior distribution and the channel, the hyper-distribution may have two different outputs which yield the same posterior distribution on the set of secrets, and in this case, there is no difference for the adversary observing both.

Indeed we can simplify the representation of a hyper-distribution matrix mergin all equal columns in a single column, and sum their outer probabilities. We also delete a posterior column when its outer probability of ocurring is 0 (which means that this "world" has 0 probability of occuring).

The map key is the index of a posterior distribution in the reduced hyper-distribution matrix.

The map value of a key i is the set of channel outputs indexes which gave rise this single posterior distribution i.

Example: If the outputs 2 and 4 from the channel yield the same posterior distribution, they are merged in, for example, the 3rd posterior distribution. Then, when you call labels[2] (the 3rd posterior has index 2), you will have the set {2,4}.
