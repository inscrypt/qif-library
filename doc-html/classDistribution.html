Distribution class

Probability distributions on sets of elements.

Contents

  • Reference
    • Public static functions
    • Constructors, destructors, conversion operators
    • Public functions
    • Public variables

A Distribution object has two attributes:

  • num_el : The number of elements in the set.
  • prob : A vector containing the probability of each element from the set.

The vector prob is valid when all the elements are in the interval [0,1] and the sum of all numbers in the vector is 1, with margin of error (upper or lower) equals to a constant epsilon, by default set as 10e-6.

Function documentation

static bool Distribution::isDistribution(std::vector<long double>& prob)

Check if a vector of numbers contains a probability distribution.

Parameters
prob A vector containing a probability distribution.
Returns It returns true if the vector contains a probability distribution or false otherwise.

A vector contains a probability distribution if:

  • For every probability p in the vector, 0 <= p <= 1;
  • The sum of all probabilities in the vector is 1 , with margin of error (upper or lower) equals to a constant epsilon, by default set as 10e-6.

Distribution::Distribution()

Default constructor.

This constructor creates a Distribution object with num_el = 0 and an empty vector prob.

Distribution::Distribution(std::string file)

Constructor used when the probability distribution is in a file.

Parameters
file File name that contains a probability distribution.

The file format must be the following:

n
p1
p2
...
pn

where n is the number of elements in the distribution and pi is the probability of the ith element occurs.

std::string Distribution::toString(int precision)

Generates a string with the probability distribution.

Parameters
precision Decimal precision for long double numbers. For example, use the value 3 to print 0.322, use 5 to print 0.32197, and so on.
Returns

A string in the following format:

p1 p2 p3 ... pn

where pi is the probability of the ith element occurring. Each two values are separated by a space (" ").

void Distribution::printToFile(std::string file, int precision)

Prints the probability distribution in a file.

Parameters
file File name to print the distribution.
precision Decimal precision for long double numbers. For example, use the value 3 to print 0.322, use 5 to print 0.32197, and so on.

The probability distribution is printed in the following format:

n
p1
p2
...
pn

where n is the number of elements in the probability distribution and pi is the probability _p(i) of the ith element occuring.

Variable documentation

int Distribution::num_el

Number of elements in the distribution.

The number of elements in the distribution is equal to prob size.

std::vector<long double> Distribution::prob

A vector of elements' probabilities.

The vector contains the probabilities of all elements from a set. Each value prob[i] is the probability of the ith element occurs. The vector is indexed from the position 0 (1st element) to the position num_el - 1 (last element). So the vector size = num_el.
