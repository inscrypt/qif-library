Channel class

Informational channels.

Contents

A channel is a system that takes as input a secret x, whose possible values come from a finite set X , and whose only observable behavior is to produce an output y, whose possible values come from a finite set Y.

A Channel object has three attributes:

  • matrix : A channel matrix from X to Y whose rows give the distribution on outputs corresponding to each possible input. That is, entry matrix[x][y] denotes p(y|x), the conditional probability of getting output y given input x. Note that all entries in a channel matrix are between 0 and 1, and each row sums to 1.
  • prior : A Distribution object, which represents the prior probability distribution on the set of secrets.
  • num_out : The number of outputs in the channel. It is exactly the number of columns in matrix.

Warning

To create a Channel object you need first to create a Distribution object, to keep the prior distribution on the set of secrets.

Function documentation

static bool Channel::isChannel(std::vector<std::vector<long double>>& matrix)

Verifies if a matrix is a valid channel matrix.

Parameters
matrix A channel matrix

A channel matrix is valid if each row is a probability distribution.

Channel::Channel()

Default constructor.

This constructor creates a Channel object with num_out = 0, prior = NULL and an empty matrix.

Channel::Channel(Distribution& prior, std::string file)

Construtor used when the channel matrix is in a file.

Parameters
prior Prior distribution on the set of secrets.
file File name that contains a channel matrix.

The file format must be the following:

n y
p11 p12 ... p1y
p21 p22 ... p2y
...
pn1 pn2 ... pny

where n is the number of secrets and y is the number of outputs. Each pij denotes p(j|i), the conditional probability of getting output j given input i. Each two numbers must be separated by a space (" ").

Warning

The number of rows in the channel matrix must be equals to prior -> num_el.

Channel::Channel(Distribution& prior, int num_out)

Constructor used to generate random channels.

Parameters
prior Prior distribution on the set of secrets.
num_out Number of outputs.

Given a set of secrets in the object prior, the constructor builds a random channel matrix. The number of the channel outputs is passed as an argument.

std::vector<std::vector<long double>> Channel::matrix

Channel matrix.

A channel matrix of a set of secrets X and a set of possible outputs Y, whose rows give the distribution on outputs corresponding to each possible input. That is, entry matrix[x][y] denotes p(y|x), the conditional probability of getting output y given input x. Note that all entries in a channel matrix are between 0 and 1, and each row sums to 1.

The rows and columns are indexed from the position 0 (1st element) to the position prior ->num_el-1 (for rows) or num_out-1 (for columns).
